@startuml MoviesSaad_Sequence_Diagram

title AI Chat Interaction (CineBot)

actor User
participant AvatarChatActivity
participant MovieAiClient
participant OllamaAPI
participant OpenRouterAPI
participant TTS_Engine

User -> AvatarChatActivity: Send Message (Voice/Text)
AvatarChatActivity -> MovieAiClient: chat(history)
MovieAiClient -> OllamaAPI: POST /chat (Local LLM)

alt Ollama Success
    OllamaAPI --> MovieAiClient: Response
else Ollama Failure
    MovieAiClient -> OpenRouterAPI: POST /chat (Fallback)
    OpenRouterAPI --> MovieAiClient: Response
end

MovieAiClient --> AvatarChatActivity: onSuccess(message)
AvatarChatActivity -> TTS_Engine: speak(text)
AvatarChatActivity -> AvatarChatActivity: Set Animation (TALKING)
TTS_Engine --> User: Voice Output
AvatarChatActivity -> User: UI Update (Chat Bubble)

@enduml
